# 股票行情数据平台深度对比 (2024-2025)

本文件对比了国内外主流的金融数据提供商，主要针对**免费版**的权限进行梳理，旨在帮助量化交易初学者或小微开发者选择合适的工具。

## 1. 核心对比表

| 平台名称 | 核心支持市场 | 免费版数据颗粒度 | 历史回溯深度 | 每日请求限制 | 接入方式 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Twelve Data** | 全球/美股/外汇 | 1min - 1mo | 日线:全历史; 1min:2020起 | 800次/天 | REST/WS |
| **AkShare** | A股/港美股/期货 | 分时, 日线, Tick | 日线:全历史; 分时:仅近期 | 无硬性限制 | Python库 |
| **Tushare Pro** | A股/基金/期货 | 日线为主 (分时需权限) | 日线:全历史; 分时:约10年 | 积分制 | API/SDK |
| **BaoStock** | A股 (沪深) | 5-60min, 日线 | 日线:1990起; 分时:约10年 | 无硬性限制 | Python库 |
| **Alpha Vantage** | 全球股票/外汇 | 日线, 历史 Intraday | 日线:20年+; 分时:最近15天 | 25次/天 | REST API |
| **Tiingo** | 美股/部分A股 | EOD (日线历史) | **日线:1962起 (60年+)** | 1000次/天 | REST API |
| **JQData (聚宽)** | A股 (沪深) | 1min, 日线 | 试用版:1年窗口 (延迟3个月) | 100万条/天 | Python库 |
| **yfinance** | 全球市场 | 1min - 日线 | 日线:全历史; 1min:仅7天 | 无官方限制 | Python库 |

## 2. 详细分析

### 2.1 国内 A 股最佳选择
*   **AkShare**: 如果你不想被积分、API Key 束缚，AkShare 是唯一的选择。它集成了东方财富、新浪财经、雪球等多个数据源。
*   **BaoStock**: 适合需要长期、大批量下载 A 股 5 分钟级 K 线进行回测的用户。

### 2.2 美股/全球市场最佳选择
*   **Twelve Data**: 适合需要多资产（外汇+美股+加密）实时监控的应用，API 设计非常现代。
*   **Tiingo**: 它的 EOD 数据质量在业内口碑极佳，1000 次/天的额度足够个人开发者构建一个精美的看板。

### 2.3 历史回溯深度详解 (Backtesting Depth)
| 周期 | 深度表现优秀的平台 | 局限性明显的平台 |
| :--- | :--- | :--- |
| **日线 (Daily)** | **Tiingo** (1962年至今); **BaoStock/Tushare** (A股1990年至今) | **JQData 试用版** (仅限中间1年且延迟3个月) |
| **分钟线 (Intraday)**| **BaoStock** (5min可达10年); **Twelve Data** (1min自2020年起) | **yfinance** (1min仅7天); **Alpha Vantage** (仅15天) |

### 2.4 量化回测推荐
*   **JQData (聚宽)**: 如果你是为了进行严肃的策略回测，聚宽的数据清洗（复权处理、财务指标修正）是免费平台中最接近收费终端的。

## 3. 技术风险提示
1. **反爬机制**: AkShare、BaoStock 和 yfinance 均依赖抓取，大规模高频访问会导致本地 IP 被暂时锁定。
2. **复权处理**: 使用数据前请务必确认是否为“前复权”或“后复权”，不同平台的处理方式不同。
3. **延迟**: 免费版 API 通常有 15-20 分钟的数据延迟（非实时）。
